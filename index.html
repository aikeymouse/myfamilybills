Quick Installation

Get up and running in minutes with Docker

1

Install Docker

Download and install Docker Desktop for your operating system from docker.com

2

Run MyFamilyBills

Copy and paste this command in your terminal:

docker run -d --name myfamilybills -p 8085:80 -p 3085:3000 -v myfamilybills_data:/app/backend/data aikeymouse/myfamilybills:latest
3

Access Your App

Open your browser and go to: http://localhost:8085
